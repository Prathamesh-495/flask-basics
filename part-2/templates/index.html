Welcome to Flask Templates!

This page is being served from an HTML file inside the templates/ folder using Flask.

Flask templates help you separate backend logic from frontend design, making your application easier to understand and maintain.

What's happening?

When you visit this URL, Flask:

  1. Receives your browser's request
  2. Matches the URL to the @app.route('/')
  3. Calls the home() function
  4. Loads templates/index.html
  5. Sends this HTML back to your browser

Why Use Templates?

  • Separation of Concerns: Python handles logic, HTML handles presentation
  • Easier to Maintain: Edit HTML without touching Python code
  • Reusability: Same template can be used with different data
  • Jinja2 Power: Add logic like loops and conditions

Common Flask Template Features

  • Displaying variables sent from Flask
  • Using conditional statements
  • Looping through lists
  • Reusing layouts
